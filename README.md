# Island Rumble

A two player game, which  players play on the same computer.
You evade opponents bullets, while trying to eliminate them!

# Dependencies

Pygame: `pip install pygame`

My module: `pip install bearsharkutils`

# Installaton

Just install the folder and dependencies on your computer.
